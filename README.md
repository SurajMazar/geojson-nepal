# GEO JSON FILE NEPAL
GEO json containing , province id , district id , municipality id and devanagarik names



### Can be used to map with data from the backend via code

## Province geojson has following properties

province_code
province_name_en
province_name_np


## District geojson has following properties

province_code
province_name_en
province_name_np
district_code
district_name_en
district_name_np

## Municipality geojson has following properties

province_code
province_name_en
province_name_np
district_code
gapa_id
municipality_code
gapa_napa
gapa_napa_np
type
key
